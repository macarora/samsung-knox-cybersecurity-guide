# Samsung Knox: A Beginner's Guide to Cybersecurity

📄 Individual technical research paper submitted at University of Newcastle

## Overview
A technical deep-dive into Samsung Knox as a mobile cybersecurity platform, 
covering cyber threats, security architecture, and real-world enterprise applications.

## Topics Covered
- Cyber threat types (Malware, Ransomware, DDoS, DNS Spoofing)
- Samsung Knox security features (Root of Trust, Knox Vault, Trusted Boot)
- Real-time Kernel Protection (RKP) — patented by Samsung
- Real-world applications including Pontifical Swiss Guard & rural banking in India
